# !getinfo

## Usage

```text
!getinfo [user]
```

## Aliases

```text
!whois
!get-info
```

## Examples

```text
!getinfo justin
!getinfo @justin
!getinfo 84117866944663552
!getinfo @justin --avatar
!getinfo @justin --avatar --groups
```

{% hint style="info" %}
This command supports flags. Available flags are:   
`username`, `id`, `avatar`, `premium`, `badges`, `groups`, `description`, `age`, `banned`, `devforum`.
{% endhint %}
