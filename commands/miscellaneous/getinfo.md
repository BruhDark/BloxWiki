# /getinfo

## Usage

```
/getinfo [user]
```

{% hint style="info" %}
This command supports flags. Available flags are:\
`username`, `id`, `avatar`, `premium`, `badges`, `groups`, `description`, `age`, `banned`, `devforum`.
{% endhint %}
