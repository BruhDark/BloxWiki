# !robloxsearch

## Usage

```text
!robloxsearch <roblox_name>
```

## Aliases

```text
!rs
!search
!roblox-search
```

## Examples

```text
!rs roblox
!rs 569422833
!rs 569422833 --username
```

{% hint style="info" %}
This command supports flags. Available flags are:  
`username`, `id`, `avatar`, `premium`, `badges`, `groups`, `description`, `age`, `banned`, `devforum`.
{% endhint %}
