# !guestrole

## Usage

```text
!guestrole <group_id> | <role> | 
<nickname> | <remove_roles>
```

## Aliases

```text
!guesbind
!guest-role
!guest-bind
```

## Permissions

**Bloxlink Manager**  
To grant this permission:  
You need the `Manage Server` role permission.
