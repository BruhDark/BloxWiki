# !verify

## Usage

```text
!verify
```

## Examples

```text
!verify add
!verify unlink
!verify view
!verify blox_link
```

{% hint style="info" %}
`!verify add` lets you link a new account.

`!verify account` lets you unlink an account.

`!verify view` shows you your linked accounts.
{% endhint %}
