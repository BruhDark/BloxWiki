# !prefix

## Usage

```text
!prefix [new_prefix]
```

## Examples

```text
!prefix ?
```

{% hint style="info" %}
Do not provide any arguments to view your current prefix.
{% endhint %}

## Permissions

**Bloxlink Manager**  
To grant this permission:  
You need the `Manage Server` role permission.
